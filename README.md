# Cuore_Grande
Ambiente interattivo multilivello per l’educazione emotiva di bambini con BES (Down, autismo, ADHD, dislessia). Offre 5 storie originali e personalizzate per favorire l'empatia. Include una modalità accessibile con tavole CAA, font ad alta leggibilità, regole sociali esplicite e note psico-pedagogiche per educatori basate su linee guida AIPD.
